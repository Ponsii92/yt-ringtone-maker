📱 Setting as Ringtone / Alarm
📱 Android — Ringtone Save the MP3 to your phone's Ringtones folder (internal storage). Go to Settings → Sound → Phone Ringtone → pick your file.
⏰ Android — Alarm Copy MP3 to the Alarms folder on your phone. Open the Clock app → set alarm → sound → choose your file.
🍎 iPhone — Ringtone Convert to .m4r and sync via iTunes/Finder, or use apps like GarageBand to import the file and set it as a ringtone directly.
⏰ iPhone — Alarm In Clock → Alarm → Edit → Sound → Pick a Song → select from your Music library (add via Files app first).
